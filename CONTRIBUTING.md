# Team Contribution Guidelines
Welcome to the Task Manager project! To maintain clean code and structure, please follow these rules:
## Branching Guidelines
1. Never commit directly to the `main` or `dev` branches.
2. Create a feature branch for your work (e.g., `feature-ui` or `feature-login`).
## Code Review Process
* Every feature must be submitted via a Pull Request (PR) targeting the `dev` branch.
* At least one team member must review and approve the PR before it is merged.
